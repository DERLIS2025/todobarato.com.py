# Todopromo.com.py

Primera versión navegable de un ecommerce marketplace multicategoría construido desde cero con Next.js App Router, TypeScript, Tailwind CSS y Zustand.

## Funcionalidades

- Home comercial con top bar, header con buscador, menú, hero, categorías, banners, oferta del día, secciones de productos y newsletter.
- PLP por categoría con breadcrumb, banner, filtros visuales, ordenamiento, vista grilla/lista y paginación mock.
- PDP con galería, variantes, cantidad, carrito, comprar ahora, wishlist, compare, beneficios, tabs y relacionados.
- Carrito persistente en localStorage con Zustand.
- Wishlist y comparador persistentes.
- Checkout visual preparado para pasarela de pago, ERP, logística y facturación.
- Páginas informativas obligatorias.
- Precios en guaraníes paraguayos con `es-PY` y `PYG`.

## Comandos

```bash
npm install
npm run dev
npm run build
```

## Despliegue

La estructura está preparada para Vercel usando Next.js App Router.
